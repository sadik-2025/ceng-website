Algorithm Visualizer

A desktop suite developed in C++ to visually trace the execution paths of various sorting algorithms, assisting students in learning complex logic.

C++ Algorithms

Technology Insights

C++ Performance: Used for its extreme efficiency. Unlike higher-level languages, C++ allows for manual memory management and direct pointer manipulation, which is vital for real-time visualization.

Algorithm Logic: This focuses on Big O Complexity. By visualizing QuickSort vs. BubbleSort, we can see the literal difference between O(n log n) and O(n²) in action.

Engineering Example: Using std::vector and custom drawing loops to refresh the visual state at 60 frames per second without leaking memory during high-frequency updates.

Custom Shell Interface

A POSIX-compliant minimalist shell interface programmed entirely in C, allowing users to pipe commands, manage background processes, and read input.

C Linux

Technology Insights

C (System Bedrock): C is the closest we get to the hardware while still being human-readable. It is used here to manage system calls and file descriptors directly.

Linux Kernel Interaction: Leveraging the POSIX standard to implement process control. This is the 'real struggle' of engineering: managing the lifecycle of programs directly through the OS.

Engineering Example: Implementing the fork() and execvp() pattern to create child processes, allowing the shell to run external programs while the parent process 'waits' for completion.
